# LockIt - Password Manager

LockIt is a modern, secure password manager for Android built with Jetpack Compose, Room, and MVVM architecture.

## Features

- **Secure Locker**: Store and manage account credentials locally.
- **Master Passkey**: Access protected by a user-defined master key.
- **PassTools**: Includes a password strength meter and a customizable generator.
- **Dynamic UI**: Supports both Light and Dark modes.
- **Privacy**: All data is stored on-device using a local Room database.
- **Multilingual**: Supports English and Polish.

## Tech Stack

- **UI**: Jetpack Compose
- **Architecture**: MVVM
- **Database**: Room
- **Image Loading**: Coil
- **Media**: Media3 / ExoPlayer
- **Language**: Kotlin

## Getting Started

1. Clone the repository.
2. Open in Android Studio (Ladybug or newer).
3. Sync Gradle and run on an emulator or physical device.
